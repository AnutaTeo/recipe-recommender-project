## Project Description

The **Recipe Recommender** is a Spring Boot web application that manages recipes and users using XML files.  
Users can add recipes, create profiles and receive personalized recommendations based on cooking skill level and preferred cuisine.

The project demonstrates the use of XML technologies such as **XSD validation, XPath queries and XSL transformations**, integrated into an intuitive web interface.
